# Позиция Арташеса: AI делает сайт из идеи

Записано: 11.06.2026

---

## Суть

Арташес считает (и практикует), что любой сайт можно создать через AI,
передав нейронке только идею. Не нужен дизайнер, не нужна верстальщик —
достаточно описать что нужно, и AI собирает рабочий результат.

---

## Что это значит для Андрея

**Не тратить энергию на то, что AI делает сам.**

Конкретно:
- Не нужно самому делать новые сайты — описать идею, AI строит
- Не нужно контролировать каждый пиксель — доверять итерации
- Роль Андрея: генератор идей + принятие финального решения
- Роль AI: реализация, правки, мобильная версия

**Правило эффективности:**
Если задача сводится к «поставить задачу нейронке» — ставить задачу, не делать руками.
Лишние действия (самому верстать, рисовать, писать код) = потеря энергии там, где AI справляется.

---

## Применение

| Ситуация | Действие |
|---|---|
| Нужен новый лендинг | Описать идею → Claude строит |
| Нужна правка на сайте | Сказать что не так → Claude правит |
| Нужен новый блок | Описать блок → Claude добавляет |
| Нужен другой сайт для другого проекта | Не делать самому → описать → AI |

Сказано в рамках обсуждения процесса разработки AI-WORKSHOP лендинга.
Арташес — практик: сам использует AI-инструменты (Claude Code, VPS, автоматизации)
