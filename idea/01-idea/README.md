# 01 — Идея

## Суть

AI Workshop — двухдневный практический тренинг, где каждый участник уходит с работающим AI-агентом под свою реальную задачу. Не лекции — руки в работе с первого часа.

Ведущий — Арташес Григорян. Не теоретик: он топ-5 брокер по продажам в Whitewheel (недвижимость, Дубай), который построил AI-системы для своего собственного бизнеса и теперь передаёт это другим.

## Почему сейчас

- AI-инструменты стали доступны без программирования — любой предприниматель может внедрить за 2 дня
- Рынок перегрет онлайн-курсами «про AI» — живой оффлайн-формат с результатом на руках выделяется
- Дубай — концентрация платёжеспособной русскоязычной аудитории предпринимателей

## Смещение Арташеса

Арташес переориентирует деятельность с брокерства на воркшопы как основное направление. Продюсер — Андрей Брезгин.
