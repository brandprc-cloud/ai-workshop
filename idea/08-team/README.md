# 08 — Команда

## Состав

| Участник | Роль | Зона ответственности |
|---|---|---|
| Андрей Брезгин | Продюсер | Стратегия, деньги, партнёры, финальные решения |
| Арташес Григорян | Эксперт / ведущий | Проведение воркшопа, экспертный контент, сеть в недвижимости |
| Яна Сахарова | Сопродюсер / методология | Программа, структура воркшопа, методические материалы |
| Анастасия | SMM | Контент, публикации, ведение соцсетей |
| Захар | Продажи | Переписка с клиентами, созвоны, доведение до оплаты |

---

## Функционал (черновик — уточняется с Андреем)

### Андрей — Продюсер
**Делает:** стратегия, партнёрства, корпоративные сделки, контроль команды, финальные решения  
**Не делает:** переписку с клиентами, создание контента, методологию  
**К нему идут:** стратегические решения, блокеры, бюджет, нестандартные условия

### Арташес — Эксперт
**Делает:** проводит воркшоп, является лицом продукта, участвует в контенте, открывает двери в сфере недвижимости  
**К нему идут:** программа, площадка, выход на контакты в недвижимости

### Яна — Методология
**Делает:** программа воркшопа, методические материалы, подготовка к каждому потоку  
**К ней идут:** вопросы по программе, учебные материалы

### Анастасия — SMM
**Делает:** ведёт соцсети, готовит контент, оформляет кейсы и отзывы  
**К ней идут:** посты, анонсы, публикация кейсов

### Захар — Продажи
**Делает:** отвечает на входящие, созванивается, доводит до оплаты, фиксирует статус лидов  
**К нему идут:** статус лида, помощь в закрытии

---

## Инфраструктура

- **Чат без Анастасии:** Андрей, Арташес, Яна, Захар
- **Общий чат:** все участники включая Анастасию
- **CRM:** есть, в доработке — Андрей разбирается
- **Бот:** выдаёт количество оплативших по дате и месту

## Регламент взаимодействия

### Общие правила
- Рабочие вопросы — в общем чате (с тегом нужного человека)
- Срочное (ответ нужен сегодня) — тегаешь напрямую
- Решения по стратегии или деньгам — только через Андрея
- Скидка / нестандартные условия — согласование с Андреем обязательно

### Цикл подготовки к воркшопу
1. **За 3+ нед:** Яна подтверждает программу, Анастасия запускает анонс
2. **За 2 нед:** Захар активно работает с базой
3. **За 1 нед:** Захар закрывает оставшиеся места, Яна готовит материалы
4. **После:** Анастасия собирает отзывы и кейсы в течение 3 дней

### Продажи
- Входящие → Захар
- Корпоратив → сначала Андрей, потом Захар
- Нестандартные условия → согласование с Андреем
