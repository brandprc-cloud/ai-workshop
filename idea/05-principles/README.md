# 05 — Принципы

## Как работаем

1. **Результат — не обещание, а факт** — участник уходит с работающим агентом, не с теорией
2. **Маленькие группы** — до 10 человек, иначе качество падает
3. **Честность про ограничения** — не берём поток, если мест меньше минимума рентабельности
4. **Команда знает свои роли** — каждый делает своё, не мешает другим
5. **Скорость принятия решений** — блокер решается в день, не неделю

## Чего не делаем

- Не снижаем цену ради заполнения группы (дешевле порога = дешевле бренд)
- Не проводим воркшоп без минимального состава (6 человек)
- Не добавляем участников сверх 10 — качество важнее выручки с потока
